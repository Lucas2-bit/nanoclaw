# Ulterior

**AI Operating System** - Multi-agent orchestration platform with personality-driven agents.

Ulterior is a COO-in-a-box: a system of coordinated AI agents that run operations, manage IP portfolios, coordinate strategy, and handle day-to-day execution across WhatsApp, Telegram, and Gmail.

## Architecture

- **14 personality-driven agents** - each modeled on a real-world visionary executor (Marshall, Munger, Lutke, Wardley, Attia, and others)
- **Multi-channel orchestration** - WhatsApp (personal), Telegram (business), Gmail
- **PACT Protocol integration** - cryptographic custody transfers between agents
- **NanoClaw runtime** - the execution engine that runs agent containers, IPC, and scheduling
- **Dashboard** - real-time command center for monitoring the entire system
