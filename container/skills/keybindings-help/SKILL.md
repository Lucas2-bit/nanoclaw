# /keybindings-help — Keyboard Shortcuts Reference

Show the user these keyboard shortcuts for the Claude Code CLI:

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message / confirm |
| `Shift+Enter` | New line in input |
| `Escape` | Cancel current operation |
| `Ctrl+C` | Interrupt current generation |
| `Ctrl+D` | Exit Claude Code |
| `Tab` | Accept autocomplete suggestion |
| `Up/Down` | Navigate message history |

## Slash Commands

Type `/` to see available commands. Common ones: `/help`, `/clear`, `/compact`, `/status`, `/capabilities`.
